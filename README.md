# ch552-pad
A 3×5 CH552-based split keyboard.

The BOM is very simple; 6 SOT123 diode (SC88) packages, two 0.1µF capacitors, two 5.1K CC resistors and a USB-C connector per side. And, of course, 36 Cherry ML switches.
